About This Page

This is a basic template you can use to launch a website using GitHub Pages. Edit the HTML, add CSS or JavaScript, and customize it however you like.

How to Use

  1. Create a new GitHub repository.
  2. Add this index.html file to the root of the repo.
  3. Go to Settings → Pages → select the main branch.
  4. Your site will be live within seconds!
